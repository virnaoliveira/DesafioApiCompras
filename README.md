# Desafio ApiCompras

Essa api foi desenvolvida para consultar produtos cadastrados, detalhar um produto com base em um id, deletar produto e cadastrar produto.

### 📋 Pré-requisitos

* Visual Studio
* Sql Server

## 🛠️ Construído com

* C#
* REST Api
* EntityFramwork Core
* .Net Core
